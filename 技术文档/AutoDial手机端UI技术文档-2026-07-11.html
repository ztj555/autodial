页面主题设置通话财库登记维护

页面与导航

页面实现功能
通话CallLogFragment记录、模式、筛选、手动拨号
录上门RegisterFragmentCRM 登记
财库StatsFragment拨号、通时、上门、趋势
设置ConnectFragment连接、拨号、主题与通知

导航支持“通话优先”和“设置优先”，只重排 Tab，不改变 Fragment 索引。

主题与组件

默认主题 sky-blue，模式 light，主色 #2B6CC4。页面卡片 14–16dp 圆角，输入框与按钮 12–14dp 圆角。

设置展开区必须使用 settingGroup、settingRow、settingDivider，不能把内部行标成 bg2。

设置页

连接状态

按钮随状态显示连接、取消、断开或重连。

云服务器管理

CloudServerSheet 支持当前节点、添加、删除、设为当前、单台/全部测速、PC 同步和网络获取。

拨号模式

DialModeSheet 展示 7 种模式及完整说明，选择后即时保存并同步通话页。

拨号动画

AnimationSheet 双列展示 11 种效果并提供预览。

主题与通知

  • 16 套主题 × 7 种亮度
  • 卡片透明度与边框
  • 导航顺序
  • 连接和登记通知
  • 上次通话提示时长

通话页

财运与财气位于标题栏右侧，固定红色和紫色。列表使用状态色条、号码脱敏、SIM 标签和时长。悬浮按钮打开手动拨号盘，点击记录打开详情面板。

财库页

今日数据红紫配色；周统计紧凑双卡；本月数据主蓝强调；上门统计 3×2 网格；七日趋势使用红橙黄绿青蓝紫。

登记页

客户称呼、手机号、顾问选择和固定来访事由。提交过程包含校验、提交中、成功/失败与恢复状态。

维护约束

  1. 保留所有 Kotlin 引用的 View ID。
  2. 功能图标使用项目矢量资源，禁止使用 emoji。
  3. BottomSheet 使用主题色，并注意 GradientDrawable 的作用域冲突。
  4. 每次提交必须通过 GitHub Actions assembleRelease。
  5. 以真机截图为最终验收依据。
