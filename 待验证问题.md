## 手机端

### 1. 录上门后统计详情弹窗

**描述**：在「录上门」页面成功登记一条客户（填写称呼、手机号、选择顾问、点击完成登记），然后切换到「统计」页，点击上门统计卡片中的数字（如「今日」或「本周」），应弹出窗口显示该时段内的客户姓名、手机号和登记时间。

**预期**：弹出 AlertDialog，显示如：
```
今日上门（1条）
1. 张三  13800138000
   ⏱ 07-04 15:30
```

**验证步骤**：
1. 打开 App → 录上门 → 完成一次登记（确保显示"✅登记成功"）
2. 切换到「统计」Tab
3. 确认「上门统计」卡片中「今日」数字为 1 或更大
4. 点击「今日」数字
5. 检查弹窗内容

**状态**：⏳ 待验证

**关联修复**（2026-07-04）：
- `RegisterFragment.saveVisitRecord()` — 登记成功时写入 `visit_records` JSON 到 SharedPreferences
- `StatsFragment.loadVisitStats()` — 读取 `visit_records` 并填充 `visitRecords` 列表
- `StatsFragment.showVisitDetail()` — 弹窗显示过滤后的上门记录
- 两处 JSON 解析的静默 catch 已改为带日志的错误处理
- 调试命令：`adb logcat -s AutoDial:D StatsFragment:D`
