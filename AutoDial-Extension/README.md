# AutoDial 浏览器扩展 v3

基于 Chrome Extension MV3，支持 JWT 云认证 + PC 直连双模拨号。

## 核心功能

1. **JWT 账号登录**：通过 REST API 登录云中继 v3，支持自动续期
2. **双模路由**：PC 直连优先（每 15s 重检），云端兜底
3. **浮动按钮**：在 CRM 页面注入可拖动的拨号/挂断按钮，8 套主题可选
4. **CRM 号码检测**：自动扫描客户手机号 + 坐席自检测
5. **拨号确认**：云端拨号后轮询等待手机确认结果

## 文件说明

| 文件 | 角色 |
|------|------|
| `manifest.json` | MV3 扩展清单，匹配 `guwen.zhudaicms.com` |
| `background.js` | Service Worker：JWT 管理 + 双模路由 + 拨号轮询 |
| `content-script.js` | 所有 frame 注入：浮动按钮 + 主题系统 + CRM 号码扫描 |
| `popup.html` + `popup.js` | 弹窗界面：登录 + 状态 + 服务器配置 |
| `icons/` | 扩展图标 |

## 安装

1. Chrome 打开 `chrome://extensions/`
2. 右上角开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `AutoDial-Extension/` 目录

## 使用

### 首次登录

1. 点击扩展图标
2. 在"服务器地址"输入云中继 v3 REST API 地址（如 `http://192.168.1.100:35441`）
3. 点击"测试服务器"验证连通性
4. 输入手机号和密码，点击"登录"
5. 登录成功后，扩展图标显示状态面板

### 注册账号

- 浏览器打开 `http://服务器IP:35441/register`（由云中继 v3 提供）
- 输入手机号 + 密码（至少 6 位）

### 日常使用（自动登录）

- 登录成功后，扩展自动保存 `refresh_token`（30 天有效）
- 每次打开浏览器自动静默续期，无需再次输入密码
- 当 CRM 页面检测到您的手机号时，自动触发静默续期

## 连接模式

弹窗状态面板显示当前连接模式：

| 状态 | 含义 |
|------|------|
| **PC 直连（局域网）** | PC 端在本地运行，通过 HTTP API 直连拨号 |
| **云端连接** | PC 不在线，通过 REST API + JWT 走云中继拨号 |
| **检测中...** | 正在检测 PC 可用性 |

## 主题切换

右键拨号按钮 → "切换主题" → 8 套主题可供选择：

dark-gold · cyber-frost · deep-space · cyberpunk · minimalist · forest-green · energetic-orange · ocean-blue

## 按钮操作

- **拨号按钮**：点击拨号，左右边缘拖动，右键打开菜单
- **挂断按钮**：拨号成功后自动显示，点击挂断，左下角拖动缩放
