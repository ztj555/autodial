# AutoDial 浏览器扩展 v4.1.0

> 最后修改：2026-06-30 21:15 | MV3 | PC直连优先 → 云端PIN兜底 | 一键登记(CRM直连) | 后台PC探测 | 姓名自动检测 | 右键同步登记列表

坐席手机号即 PIN，打开 CRM 页面自动检测，无需登录，无需密码。

## 核心功能

1. **自动检测 PIN**：打开 CRM 页面 → CSS 选择器 `.user-phone` → 自动存为 PIN
2. **自动检测姓名**：同步抓取 `.user-name`，无需手动输入业务员姓名
3. **双模路由**：PC 直连优先（2s 超时），不可达时走云端 PIN 验证
4. **PC_CONNECTED 反向兜底**：云端发现 PC 在线 → 提示切回本地
5. **浮动按钮**：CRM 页面注入可拖动的拨号/挂断按钮，8 套主题
6. **右键菜单**：主题切换 / 拨号 / 短信 / 一键登记 / PC 状态 / PIN 显示 / 同步登记列表
7. **一键登记**：CRM 详情页自动识别客户姓名+手机号 → 右键确认弹窗 → CRM 直连提交（姓名→kid 两步）
8. **同步登记列表**：主管右键抓取 CRM 来访列表全量记录推入云端，补全网页端登记数据

## 文件说明

| 文件 | 角色 |
|------|------|
| `manifest.json` | MV3 扩展清单，匹配 zhudaicms.com / rxhcrm.com / rongxinhui.com |
| `background.js` | Service Worker：PIN 管理 + 双模路由 + 一键登记 API 调用 |
| `content-script.js` | 所有 frame 注入：TreeWalker 扫号 + 姓名识别 + 浮动按钮 + 主题 + 右键菜单 + 确认弹窗 |
| `popup.html` + `popup.js` | 弹窗界面：服务器配置 + PIN 设置 + 状态查询 |

## 安装

1. Chrome/Edge 打开 `chrome://extensions/`
2. 右上角开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `AutoDial-Extension/` 目录

## 使用

### 首次配置

1. 点击扩展图标
2. 设置云中继地址（默认 `http://127.0.0.1:35430`，点击"测试"验证 `/health` 连通性）
3. 打开 CRM 页面（guwen.zhudaicms.com）→ 插件自动检测坐席手机号
4. 弹出框显示检测到的号码 → 点击"保存"即完成 PIN 设置

> 需要先启动云中继：`python cloud_relay_v2.py`。见 `cloud-relay/python/`。

### 日常使用

- 打开 CRM → 自动检测坐席号 → 点击拨号按钮 → 自动选择 PC/云端路由
- PIN 状态在右键菜单底部可见
- 换号时打开 CRM 页面重新检测即可

### PIN 管理

- **自动检测**：打开 CRM 页面，content-script 自动扫描坐席手机号 → 存为 `self_phone`
- **手动设置**：弹出框输入 11 位手机号 → 点击"保存"
- **清除**：弹出框点击"清除 PIN"

## 拨号反馈

| 提示 | 原因 |
|------|------|
| `✓ 已拨出` | PC 直连成功 |
| `✗ PC已上线，请重试` | 云端发现 PC 在线，应切回本地 |
| `✗ 服务器不可达 (端口35430)` | 云中继未启动 |
| `✗ 未检测到坐席手机号` | 请打开 CRM 页面 |
| `✗ 短信仅支持 PC 直连模式` | PC 离线，短信无云端转发 |

## 右键菜单

- **🎨 切换主题**：8 套主题实时切换
- **📞 拨打 / 💬 发短信**：对当前检测到的客户号码操作
- **📝 一键登记**：客户详情页可用，自动识别姓名+手机号，弹窗确认后提交
- **🖥 打开电脑端主界面 / 📋 显示隐藏悬浮窗**：PC 端控制
- **底部状态**：`👤 PIN: 13800138000` + PC 在线状态

## 一键登记流程

1. 打开 CRM 客户详情页 → 自动识别「姓名」和「手机号」
2. 右键悬浮按钮 → 点击「📝 一键登记 张三 139xxxx」
3. 确认弹窗显示：客户姓名、手机号、顾问手机号、事由（贷款咨询）
4. 点击「确认登记」→ 云中继存储 → 后台同步 CRM → WS 推送给手机
5. 浮动按钮显示「✅ 已登记 张三」

> 需云中继在线（端口 35430）。如未连接云端，提示「无法连接云端，请先启动云中继服务」。
