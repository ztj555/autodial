# AutoDial 部署指南

> 腾讯云 101.34.65.254 | Ubuntu 22.04 | 1Panel v2.0.15 | 2026-07-18

## 一、云中继部署

### 环境要求
- Ubuntu 22.04+
- Python 3.10+
- pip: `websockets pystray Pillow`

### Docker 部署（推荐）

```bash
docker run -d --restart=always --name autodial-relay \
  -p 35430:35430 \
  -v /opt/autodial/data:/app/data \
  autodial/cloud-relay
```

### Supervisor 部署

```bash
# 安装依赖
cd /opt/autodial/cloud-relay/python
pip install websockets pystray Pillow

# Supervisor 配置 /etc/supervisor/conf.d/autodial.conf
[program:autodial-relay]
command=/usr/bin/python3 /opt/autodial/cloud-relay/python/cloud_relay_v2.py
directory=/opt/autodial/cloud-relay/python
autostart=true
autorestart=true
user=root
stdout_logfile=/var/log/autodial-relay.log
stderr_logfile=/var/log/autodial-relay-err.log

# 启动
supervisorctl reread && supervisorctl update && supervisorctl start autodial-relay
```

### 验证
```bash
curl http://localhost:35430/health
# 预期: {"ok": true, "uptime": ...}
```

## 二、双实例部署（35430 + 35440）

1Panel 运行两个容器实例，或 Supervisor 运行两个进程（不同端口）：
```bash
python cloud_relay_v2.py --port 35430 &
python cloud_relay_v2.py --port 35440 &
```

## 三、Android APK 构建

GitHub Actions 自动构建，产物包括：
- `app-release` — 正式签名 APK
- `app-debug` — Debug 签名 APK

本地构建：
```bash
cd android-app
./gradlew assembleRelease  # 需密钥
./gradlew assembleDebug    # 无需密钥
```

### 签名密钥
- 文件：`android-app/autodial-release.p12`
- 算法：RSA 2048, SHA256
- 有效期：25 年
- Secrets 配置：`KEYSTORE_BASE64` 放在 GitHub Actions Secrets

## 四、Chrome 扩展分发

1. 加载已解压的：`chrome://extensions/` → 开发者模式 → 加载 `AutoDial-Extension/`
2. 打包 .crx：扩展管理页 → 打包扩展 → 选择目录
3. 私钥文件 `.pem` 妥善保管，用于后续版本更新
